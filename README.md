# Medicine Catalogue Website

A Django-based web application for managing and displaying a catalogue of medicines.

The platform allows users to browse medicines with detailed information, images, and MRP, while providing an admin-friendly backend for managing products.  
This project focuses on clean UI, structured backend logic, and real-world usability, making it suitable for small pharmacies or medicine distributors.

---

##  Features

- Medicine listing with detailed view  
- Image support for medicines  
- Clean and responsive UI  
- Admin panel for managing medicines  
- Organized and scalable Django project structure  

---

##  Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, Tailwind CSS  
- **Database:** SQLite  
- **Tools:** Django ORM, Django Admin  

---

##  Setup & Installation

```bash
git clone https://github.com/aimanXcodes/medicine_catalogue_website.git
cd medicine_catalogue_website
python -m venv env
env\Scripts\activate   # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
# open in browser
http://127.0.0.1:8000/  

 # admin panel
 http://127.0.0.1:8000/admin/ 
 - **From here you can add, update, or remove medicines and manage uploaded images.**
